# Student Performance Analysis End-to-End

## Project Overview

This project analyzes student performance factors using Excel, SQL Server, Python, and Power BI. The objective is to identify the key factors influencing exam scores and visualize insights through interactive dashboards.

## Dataset Information

The dataset contains 6,607 student records with attributes such as:

* Hours Studied
* Attendance
* Motivation Level
* Family Income
* School Type
* Teacher Quality
* Parental Involvement
* Exam Score

### Power BI

* Interactive Dashboard
* KPI Cards
* Charts and Visualizations

## Key Insights

* Students with high motivation generally achieved higher exam scores.
* Attendance showed a positive relationship with academic performance.
* Most students were classified in the Good performance category.
* Family income had a moderate impact on exam scores.
* Excellent performers represented a small percentage of the dataset.
